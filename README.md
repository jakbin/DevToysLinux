## How to install
```sh
pip3 install devtoys
```

## How to use
```sh
devtoys format {filename.json}
devtoys convert {filename.json}
```

## Tools are available :

- Converters
    - Json <> Yaml
- Formatters
    - Json
- Encoders / Decoders
    - Base64
    - Html
    - Url
